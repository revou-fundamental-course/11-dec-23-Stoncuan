untuk menghungukan ke git repositroy maka caranya
1. git init
2. git add .
3. git remote add origin https://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1~@github.com/Stoncuan/Belajar.git
    untuk dipakai menghungunkan ke repositroy yang kita miliki dan bisa juga dipakai untuk login
    dengan manggunakan token
4. git remote -v (untuk melihat sudah terhubung atau tidak)
5. agar filenya bisa update silahkan git add .
6. setelah itu masukan perintah git commit -m "push" (push adalah nama commit ketika di commit ke git repositroy)
7. git push origin master (filenya akan langsung di push ke git repositroy yang sudah di buat)
